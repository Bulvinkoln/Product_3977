Некоторые команды для работы на Git и GitHub:

Для того, чтобы вносить какие либо изменения в чужом удаленном репозитории, этот репозиторий сначала нужно "форкнуть" (fork).

git push- это команда для переноса изменений с локального репозитория на удаленный репозиторий. 

git pull- команда для "стягивания" каких-либо новых изменений с удаленного репозитория на локальный (не зависит от того, кто и откуда внес эти изменения).
